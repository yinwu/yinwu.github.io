python map()方法
------------------------------------

map() 会根据提供的函数对指定序列做映射。
以参数序列中的每一个元素调用 function 函数，返回包含每次 function 函数返回值的新列表。

map函数的语法::

	map(function, iterable, ...)

返回值

	Python 2.x 返回列表。

	Python 3.x 返回迭代器。
